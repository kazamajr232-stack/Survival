# 🔥 Survival Hari Ini

Game simulasi manajemen waktu dan pencegahan burnout berbasis kartu untuk GitHub Pages.

## 🎮 Cara Bermain

1. Buka [GitHub Pages URL Anda]
2. Klik "MULAI SIMULASI"
3. Pilih kartu aktivitas untuk mengisi 16 jam waktu Anda
4. sesuatu akan muncul dan mengubah hal, tentukan dengan bijak
5. Klik "SELESAIKAN HARI" untuk melihat skor akhir

## 🎯 Tujuan

- Selesaikan tugas tanpa mengalami **BURNOUT**
- Kelola energi dan stress dengan bijak
- Dapatkan skor setinggi mungkin!
